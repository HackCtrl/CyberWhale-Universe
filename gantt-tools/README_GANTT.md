# Gantt tools

Коротко: инструменты для безопасной работы с `Диаграмма Ганта.xlsx`.

Файлы:
- `scripts/import_gantt.js` — импортирует первую таблицу из `Диаграмма Ганта.xlsx` в `gantt_state.json` в корне репозитория (делает резервную копию, если файл уже есть).
- `scripts/prune_to_gantt.ps1` — (Windows PowerShell) создаёт резервную копию всего рабочего каталога и затем удаляет все файлы/папки кроме `Диаграмма Ганта.xlsx`, `gantt-tools`, `.git` и `.github`. Скрипт требует подтверждения и не выполняет удаление без вашего согласия.

Установка и запуск:

1. Перейдите в папку `gantt-tools`:

```powershell
cd "c:\Users\Кирилл\Desktop\Проекты и сторонние сайты\CyberWhale Universe\gantt-tools"
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите импорт (создаст `gantt_state.json` в корне репозитория):

```bash
npm run import:gantt
```

4. (Опционально) Если вы подтверждаете очистку репозитория, откройте PowerShell в корне репозитория и выполните:

```powershell
.\gantt-tools\scripts\prune_to_gantt.ps1
```

— скрипт создаст резервную копию `workspace_backup_YYYYMMDD_HHMMSS` и затем удалит файлы после вашего подтверждения.

Примечания безопасности:
- Сначала запустите `npm run import:gantt` и проверьте `gantt_state.json`.
- Не запускайте `prune_to_gantt.ps1`, пока не убедитесь, что резервная копия создана и корректна.
