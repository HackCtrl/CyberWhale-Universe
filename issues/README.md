Как пользоваться карточками в `issues/`

- Файлы в этой папке — готовые markdown‑карточки для импортирования в GitHub Issues или ваш трекер.
- Каждый файл содержит id, epic, приоритет, estimate и чеклист микрозадач.
- Чтобы создать Issue в GitHub: скопируй содержимое файла в новое Issue или используй API/скрипты.
