CyberWhale MVP — minimal starter

Цель: чистая рабочая база для быстрого старта разработки MVP.

Что сделано:
- Минимальный каркас Next.js (app dir) + TypeScript + Tailwind

Важное правило текущего задания:
- Пункты 1.3.1 и 1.3.2 (создание репозитория и настройка CI/CD) оставляем как есть — не трогаем и не автоматизируем сейчас.
- Пункты 1.3.3 и 1.3.4 откладываем на финал проекта.

Как начать:

1. Перейдите в `mvp-starter` и установите зависимости:

```bash
cd mvp-starter
npm install
```

2. Запуск в режиме разработки:

```bash
npm run dev
```

Если хотите, могу автоматически создать простую GitHub Actions workflow для CI позже, но по текущему указанию CI оставляем без изменений.
