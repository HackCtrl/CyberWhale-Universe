Этот каталог хранит журнал прогресса задач.

Использование:
- Обновить прогресс задачи:
  node scripts/update_progress.js --id 1.2.3 --percent 35

- Файл лога: `progress_log.json` — массив записей { timestamp, id, old, new }.

Правила:
- Скрипт автоматически ограничивает значение процента по стадиям: 1 -> 35%, 2 -> 65%, 3 -> 85%, 4 -> 100%.
- Все изменения сохраняются в `gantt_state.json`.
