Prototype — интерактивный статический макет

Файлы:
- docs/prototypes/index.html — Главная
- docs/prototypes/dashboard.html — Личный кабинет
- docs/prototypes/courses.html — Каталог/карточки курсов
- docs/prototypes/ctf.html — Страница CTF
- docs/prototypes/style.css — Стили прототипа

Как открыть:
1. Откройте файл `docs/prototypes/index.html` в браузере (двойной клик или "Открыть с помощью" → браузер).
2. Навигация между страницами реализована ссылками (кликабельные элементы).

Примечание:
- Это статический прототип: клики ведут между страницами, нет серверной логики.
- Если нужно разместить онлайн — могу помочь с простым GitHub Pages / Netlify deployment.
