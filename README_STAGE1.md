# CyberWhale Universe — Stage 1 (Week 1)

Коротко: в этой неделе фокус на подготовке репозитория, документации и базовой инфраструктуре для разработки MVP.

Что включено (минимум):
- Основной репозиторий на GitHub (создан)
- CI workflows для backend и frontend (`.github/workflows/*`)
- Документация: `docs/PROGRESS_UPDATE.md`, `docs/DB_SCHEMA.md`
- Базовая структура `apps/backend`, `mvp-starter`

Задачи на неделю 1 (приоритет):
1. Подготовить репозиторий: README, CODEOWNERS, ISSUE_TEMPLATE, метаданные — выполнено частично.
2. Утвердить схемы БД и создать миграции (см. `docs/DB_SCHEMA.md`).
3. Запустить CI и убедиться в успешных билд-пайплайнах.
4. Создать первую страницу UI "Чем займёмся сегодня?" и скрипт выбора задачи.

Как отмечать прогресс: следуйте `docs/PROGRESS_UPDATE.md`.
